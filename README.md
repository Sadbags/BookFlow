**Bookflow** is a web platform for book lovers, inspired by Letterboxd, where users can discover, organize, and track their reading activity.

Users can:

* Mark books as read
* Save books to read later
* Add books to favorites
* Keep a personal record of their reading history
* Discover new books through the community

The application is built as a full-stack project, featuring user authentication, reading status management, and a user-focused experience for organizing and tracking books.
